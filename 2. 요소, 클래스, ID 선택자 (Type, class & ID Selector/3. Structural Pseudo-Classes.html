Structural Pseudo-classes (구조적 가상 클래스) element:first-Child element:last-child element:nth-child(n) ------------------------------------------------

Structural Pseudo-classes

  1. Do something
  2. do something
  3. Create something
  4. Create something else
  5. Make something
  7. Make something else
ex) CSS li:frist-child { color: #0066FF; } li:last-child { color: #ffc82c } li-nth-child(3) { // 숫자가 위치를 나타냄 혹은 2n = 짝수 2n-1 홀수 로 지정할 수도 있다. color: #ff4949 }
